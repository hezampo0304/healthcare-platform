Healthcare Platform is an enterprise-grade Healthcare Management System (HMS) designed using a microservices architecture.

The platform aims to provide a complete digital ecosystem for hospitals, clinics, medical centers and pharmacies, enabling secure and scalable management of patients, doctors, appointments, medical records, pharmacy inventory, billing and reporting.

This project follows modern software engineering practices including:

• Domain-Driven Design (DDD)
• Clean Architecture
• Hexagonal Architecture
• Event-Driven Architecture
• SOLID Principles
• Test-Driven Development (TDD)
• CI/CD
• Observability
• Cloud-Native Design

The goal is to simulate a real enterprise environment using technologies commonly adopted in large organizations.
